# mmo-project
java server, phaser.js client

# How to run

## Server

Windows powershell run: (requires jdk23)

```sh

mvn compile exec:java -D"exec.mainClass=mmo.Main"

```


## Client

Requires NodeJs and Phaser.
(`npm install phaser`)

```sh
cd client

npm run dev
```
